¿Qué es pip y por qué lo actualizamos?

pip es el instalador de paquetes de Python. Lo usamos para instalar bibliotecas externas necesarias para nuestros proyectos (como Flask, Django, NumPy, etc.).
Se actualiza para asegurarnos de contar con la última versión, que incluye mejoras de seguridad, compatibilidad y acceso a más paquetes.
